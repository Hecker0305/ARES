# Standards Reference

## MISP Attribute Categories
- **Internal reference**: Internal reference data for the organisation
- **Antivirus detection**: AV signature detection information
- **Payload delivery**: Delivery mechanism of the payload (e.g., email, USB)
- **Artifacts dropped**: Files dropped on the system by the payload
- **Payload installation**: Installation vectors of the payload
- **Persistence mechanism**: System changes to maintain persistence
- **Network activity**: Network traffic indicators
- **Payload type**: Type of payload (RAT, dropper, downloader)
- **Attribution**: Evidence for adversary attribution
- **External analysis**: Third-party analysis reports
- **Financial fraud**: Financial fraud indicators

## MISP Distribution Levels
| Level | Description |
|-------|-------------|
| 0 | Your organisation only |
| 1 | This community only |
| 2 | Connected communities |
| 3 | All communities |
| 4 | Sharing group |

## MISP Threat Levels
- 1: Sophisticated, APT-level threat
- 2: Advanced persistent threat
- 3: Opportunistic threat
- 4: Unclassified
