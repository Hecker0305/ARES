# Standards Reference

## MITRE ATT&CK Mappings
- T1525: Cloud Infrastructure Discovery - Azure resource enumeration
- T1613: Container and Resource Discovery - Azure container registry discovery
- T1530: Data from Cloud Storage - Azure Blob storage data exposure

## NIST CSF Mappings
- DE.CM-1: Continuous monitoring of cloud resources via Defender for Cloud
- ID.AM-1: Asset inventory across Azure subscriptions
- PR.AC-1: Azure AD managed identities and conditional access
- PR.DS-5: Azure Storage Service Encryption and SQL TDE

## Azure Security Benchmark v3.0
- NS-1: Implement network segmentation boundaries
- IM-1: Use Azure AD as the default authentication method
- PA-1: Separate and limit highly privileged users
- DP-1: Data classification and protection
- LT-1: Enable logging for all Azure resources

- SOC 2 Type II attestation support
