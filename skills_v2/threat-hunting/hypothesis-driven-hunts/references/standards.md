# Standards Reference

## MITRE ATT&CK Mappings
- T1059: Command and Scripting Interpreter - Hunting for suspicious script execution
- T1068: Exploitation for Privilege Escalation
- T1078: Valid Accounts - Hunting for anomalous authentication patterns
- T1547: Boot or Logon Autostart Execution - Persistence hunting
- T1562: Impair Defenses - Tampering with security tools

## Hunt Hypothesis Framework
- **SARA**: Situation, Adversary, Response, Action
- **Diamond Model**: Adversary, Infrastructure, Capability, Victim
- **Cyber Kill Chain**: Recon -> Weaponize -> Deliver -> Exploit -> C2 -> Actions

## NIST CSF Hunting Mappings
- DE.AE-1: Event data is collected and correlated
- DE.AE-2: Event analysis identifies attack patterns
- DE.AE-3: Event data is analyzed with intelligence
- DE.CM-1: Network monitoring for unauthorized activities
