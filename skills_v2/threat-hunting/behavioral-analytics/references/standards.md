# Standards Reference

## MITRE ATT&CK Mappings
- T1078: Valid Accounts - Anomalous login patterns indicating credential abuse
- T1098: Account Manipulation - Unusual privilege changes or group modifications
- T1136: Create Account - Suspicious account creation events
- T1531: Account Access Removal - Abnormal account lockout patterns

## Behavioral Analytics Frameworks
- NIST SP 800-207: Zero Trust Architecture (UEBA as continuous verification)
- Gartner UEBA Market Guide: Key behavioral analytics capabilities
- SANS Insider Threat Detection: Behavioral indicators of insider threats

## Analytical Methods
- Z-score: (x - μ) / σ for normally distributed features
- Modified Z-score: 0.6745 * (x - median) / MAD for robust outlier detection
- Isolation Forest: Ensemble method for anomaly detection
- Local Outlier Factor (LOF): Density-based anomaly detection
- Seasonal-trend decomposition: Time series decomposition for periodic patterns
