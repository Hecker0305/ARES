# Phishing Incident Response Standards

## Frameworks and Standards
- NIST SP 800-61 Rev 2 (Computer Security Incident Handling Guide)
- NIST SP 800-83 (Guide to Malware Incident Prevention and Handling for Desktops and Laptops)
- FIRST PSIRT Services Framework
- APWG Phishing Activity Trends Reports
- DMARC.org Specification
- RFC 7489 (DMARC), RFC 6376 (DKIM), RFC 7208 (SPF)

## Recommended Tools
- PhishTool, URLScan.io, VirusTotal, InQuest
- M365 Defender / Proofpoint / Mimecast admin consoles
- any.run / Joe Sandbox for attachment analysis
- SIEM/SOAR platforms (Splunk SOAR, Palo Alto XSOAR, Microsoft Sentinel)
